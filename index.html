About

Clinicians querying oncology trial repositories frequently need ad-hoc, multi-constraint questions over biomarkers, endpoints, interventions, and time. Writing SQL directly requires schema expertise and slows analysis.

FD-NL2SQL is a feedback-driven clinical NL2SQL assistant. It decomposes a natural-language query into predicate-level sub-questions, retrieves semantically similar expert-verified exemplars, and synthesizes executable SQL grounded in schema context.

The system improves with use by learning from approved user edits and logic-based SQL augmentations, automatically expanding its exemplar bank for better future generations.
